# Tobin at Large

Personal site for Kyle J. Tobin. Writing, consulting, and AI architecture.

## Setup

Single-page static site. No build step, no dependencies.

```
git clone <repo>
open index.html
```

## Deploy

GitHub Pages. Push to `main` and it's live.

## Design System

`DESIGN.md` contains the full visual spec: parchment palette, typography rules, component guidelines. Hand it to Claude Design or any design agent alongside the reference assets to generate on-system pages.

## Assets

- `tal-mark.svg` — TAL monogram logo
- `tal-logo.png` — Logo raster version
- `cover.png` — Book cover (reference only, not displayed on current site)
- `DESIGN.md` — Design system spec
